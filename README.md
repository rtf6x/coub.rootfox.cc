# Coub Player

Custom coub-like player: looped video, separate audio, playlists.

- Site: https://coub.rootfox.cc
- Media: https://media.rootfox.cc/coub/

Push to `main` → CI deploys site and syncs new videos from `src/videos/` to R2.

GitHub secrets: `CLOUDFLARE_API_TOKEN`, `CLOUDFLARE_ACCOUNT_ID`, `R2_ACCOUNT_ID`, `R2_ACCESS_KEY_ID`, `R2_SECRET_ACCESS_KEY`, `R2_BUCKET`.
